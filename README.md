# 🤖 IA Text Summarizer - Proyecto de Julio

Mi primer proyecto de Inteligencia Artificial utilizando **Python** y modelos de **Hugging Face**. Este script utiliza modelos de lenguaje pre-entrenados para resumir textos largos de manera automática.

## 🚀 Funcionalidades
- Resumen inteligente de textos.
- Uso del modelo `distilbart-cnn-12-6` (Optimizado para velocidad).
- Procesamiento de Lenguaje Natural (NLP).

## 🛠️ Tecnologías
- **Lenguaje:** Python 3.12+
- **Librerías:** Hugging Face Transformers, PyTorch.
- **Modelo:** DistilBART.

## 📦 Instalación
Para ejecutar este proyecto en otro entorno:
1. Clonar el repo.
2. Ejecutar `pip install -r requirements.txt`.
3. Correr con `python app.py`.
